# Wikinotes

Documentation technique personnelle d'Alan Marchand. Ce wiki rassemble mes
notes de référence sur l'administration système, le développement Python,
l'électronique embarquée, le mainframe z/OS et la sécurité.

Chaque onglet ci-dessous correspond à un domaine. Utilisez la barre de
navigation en haut de page ou la recherche pour accéder directement à une
fiche.

<div class="grid cards" markdown>

-   :material-ubuntu: **[Ubuntu](systeme/ubuntu/index.md)**

    Post-installation, dotfiles `alm_dots` et boîte à outils `alm_tools`.

-   :material-linux: **[Alpine](systeme/alpine/index.md)**

    Alpine Linux : machines virtuelles, partages de répertoires et Docker.

-   :material-language-python: **[Développement](developpement/index.md)**

    Python moderne : `uv`, concurrence, FastAPI et design patterns.

-   :material-chip: **[Électronique](electronique/index.md)**

    Développement embarqué avec PlatformIO et le générateur `pioinit`.

-   :material-tools: **[Outils](outils/index.md)**

    Kitty, yt-dlp, Claude Code et MkDocs.

-   :material-mainframe: **[LCL](lcl/index.md)**

    Mainframe z/OS : Kafka, CICS et le plugin zDevOps pour IDz.

-   :material-shield-lock: **[Sécurité](securite/index.md)**

    TLS, AppArmor, OpenSSL, chiffrement eCryptfs et Proton.
