# Environnement VS Code Docker — Zowe & Copilot

Environnement de développement isolé sous Docker pour macOS Apple Silicon (M4).  
Intègre VS Code (code-server), Zowe CLI, GitHub Copilot et les runtimes Java, Node.js et Python.  
Image construite par l'équipe support et déployée via SCCM.
