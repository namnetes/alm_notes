# alm-tools

`alm-tools` automatise la mise en place d'un environnement de développement complet sur Ubuntu 24.04. Son cœur est un script de post-installation en 20 étapes qui configure un poste entier en une seule commande.

**Dépôt :** [github.com/namnetes/alm-tools](https://github.com/namnetes/alm-tools)
