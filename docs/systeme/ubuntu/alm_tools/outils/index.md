# Outils spécialisés

Outils CLI et scripts utilitaires du dépôt `alm_tools`. Chaque carte indique
le rôle de l'outil ; sa source dans le dépôt est précisée sur sa page.

<div class="grid cards" markdown>

-   :material-language-python: **[devinit](devinit.md)**

    Scaffolder de projets Python (source : `cli/devinit/`).

-   :material-book-cog-outline: **[mkdocsinit](mkdocsinit.md)**

    Configuration MkDocs Material idempotente (source : `cli/mkdocsinit/`).

-   :material-chip: **[pioinit](../../../../electronique/pioinit/index.md)**

    Scaffolder de projets PlatformIO (source : `cli/pioinit/`).

-   :material-server: **[vmforge](vmforge.md)**

    Automatisation de VMs KVM (source : `cli/vmforge/`).

-   :material-web: **[open-sites](open-sites.md)**

    Lanceur de pages de connexion à des sites, via `fzf` (source : `cli/open-sites/`).

-   :material-cog-play-outline: **[runbg](runbg.md)**

    Lanceur de scripts en arrière-plan (source : `jobs/runbg.sh`).

-   :material-google-drive: **[backup_googledrive](backup-googledrive.md)**

    Sauvegarde Google Drive avec déduplication
    (source : `jobs/backup_googledrive.sh`).

-   :material-source-branch-remove: **[reset_historique_git](reset-historique-git.md)**

    Écrase l'historique Git d'un dépôt en un seul commit, avec sauvegarde
    préalable (source : `jobs/reset_historique_git.sh`, raccourci `grh`).

</div>
